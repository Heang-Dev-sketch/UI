Style Your Web: CSS

Learn Cascading Style Sheets to bring your web pages to life with colors, layouts, and fonts.

1. Introduction to CSS

CSS (Cascading Style Sheets) is used to style and lay out web pages. You can change colors, fonts, spacing, and more.

<!DOCTYPE html>
<html>
<head>
<style>
body {
  background-color: lightblue;
}
h1 {
  color: white;
  text-align: center;
}
</style>
</head>
<body>

<h1>My First CSS Example</h1>
<p>This is a paragraph.</p>

</body>
</html>
