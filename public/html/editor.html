How to Get Started

1. Write HTML: The left panel is for your HTML structure. Think of it as the skeleton of your webpage. Try adding a new tag like <h2>My Second Heading</h2>.

2. Add CSS: The right panel is for CSS styles. This is how you add color, layout, and fonts. Try styling your new heading: h2 { color: deepskyblue; } in the CSS panel.

3. See the Result: Click the "Run Code" button. Your changes will instantly appear in the "Live Preview" panel above. Experiment and have fun!
