Master the Foundation: HTML

Start your web development journey by learning the structure of every webpage.

1. Introduction to HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page using elements.

<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>
    <h1>My First Heading</h1>
    <p>My first paragraph.</p>
</body>
</html>

2. Headings & Paragraphs

Headings are defined with `<h1>` to `<h6>` tags, while paragraphs use the `<p>` tag.

<h1>This is heading 1</h1>
<h2>This is heading 2</h2>
<p>This is a paragraph of text. It can contain multiple sentences.</p>

3. Links (Anchor Tags)

The `<a>` tag defines a hyperlink, which is used to link from one page to another. The `href` attribute indicates the link's destination.

<a href="https://www.google.com">Visit Google</a>

4. Images

The `<img>` tag is used to embed an image in an HTML page. The `src` attribute specifies the path to the image, and `alt` provides alternate text.

<img src="https://via.placeholder.com/150" alt="A 150x150 placeholder image">

5. Lists (Ordered & Unordered)

HTML supports ordered (numbered) and unordered (bulleted) lists.

<h3>Unordered List</h3>
<ul>
  <li>Coffee</li>
  <li>Tea</li>
</ul>

<h3>Ordered List</h3>
<ol>
  <li>First item</li>
  <li>Second item</li>
</ol>

6. Tables

HTML tables allow you to arrange data into rows and columns.

<table border="1">
  <tr>
    <th>Firstname</th>
    <th>Lastname</th>
  </tr>
  <tr>
    <td>Peter</td>
    <td>Griffin</td>
  </tr>
</table>

7. Forms

The `<form>` element is used to create an HTML form for user input.

<form>
  <label for="fname">First name:</label><br>
  <input type="text" id="fname" name="fname"><br>
  <label for="lname">Last name:</label><br>
  <input type="text" id="lname" name="lname"><br><br>
  <input type="submit" value="Submit">
</form>

8. Semantic HTML

Semantic elements like `<header>`, `<nav>`, `<main>`, `<section>`, and `<footer>` clearly describe their meaning to both the browser and the developer.

<header>
  <h1>Website Title</h1>
</header>
<nav>
  <a href="#">Home</a> | <a href="#">About</a>
</nav>
<main>
  <p>Main content here.</p>
</main>
<footer>
  <p>Copyright 2025</p>
</footer>

9. Div and Span

The `<div>` element is a block-level container for other HTML elements, while `<span>` is an inline container used to mark up a part of a text, or a part of a document.

<div style="background-color: lightgrey; padding: 15px;">
  <h2>This is a div</h2>
  <p>This is some text with a <span style="color:blue;font-weight:bold;">span</span> inside.</p>
</div>

10. Comments

HTML comments are not displayed in the browser, but they can help document your HTML source code.

<!-- This is a comment -->
<p>This is a paragraph.</p>
<!-- Comments are not displayed in the browser -->
